# Lattice 格致 - 快速上手

## 下载安装

### Windows

1. 下载 [Lattice_0.2.0_x64-setup.exe](https://github.com/tryandaction/lattice/releases/latest)
2. 双击运行安装程序
3. 安装完成后，从开始菜单或桌面快捷方式启动

### macOS / Linux

即将推出，敬请期待。

## 基本使用

### 打开文件夹

1. 点击左侧边栏的「Open Local Folder」按钮
2. 选择包含你的研究文件的文件夹
3. 文件树会显示在左侧边栏

### 支持的文件格式

| 格式 | 说明 |
|------|------|
| PDF | 阅读、高亮、批注 |
| Markdown | 编辑、预览 |
| Jupyter Notebook | 查看、运行 Python |
| PowerPoint | 预览 |
| Word | 预览 |
| 图片 | PNG, JPG, GIF 等 |
| 代码 | Python, JavaScript 等 |

### PDF 批注

1. 打开 PDF 文件
2. 选中文字进行高亮
3. 点击高亮区域添加评论
4. 批注自动保存

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 切换侧边栏 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+W` | 关闭当前标签 |

## 常见问题

### 应用无法启动

1. 确保 Windows 10 1803 或更高版本
2. 尝试以管理员身份运行
3. 检查是否有杀毒软件拦截

### 文件无法打开

1. 确保文件格式受支持
2. 检查文件是否损坏
3. 尝试重新打开文件夹
