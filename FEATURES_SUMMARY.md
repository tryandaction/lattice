## ⌨️ 快捷键

### 通用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+B / Cmd+B | 切换侧边栏 |
| Ctrl+S / Cmd+S | 保存文件 |
| Ctrl+O / Cmd+O | 打开文件 |
| Ctrl+W / Cmd+W | 关闭标签页 |
| Ctrl+Tab | 切换标签页 |

### 桌面应用专属

| 快捷键 | 功能 |
|--------|------|
| Ctrl+, / Cmd+, | 打开设置 |

### 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z / Cmd+Z | 撤销 |
| Ctrl+Y / Cmd+Y | 重做 |
| Ctrl+F / Cmd+F | 查找 |
| Ctrl+H / Cmd+H | 替换 |
| Ctrl+/ / Cmd+/ | 注释/取消注释 |

### 数学编辑快捷键

| 按键 | 功能 |
|------|------|
| / | 创建分数 |
| ^ | 创建上标 |
| _ | 创建下标 |
| Tab | 下一个占位符 |
| Shift+Tab | 上一个占位符 |
