# Obsidian Compat Extended (Example)

This example demonstrates create/rename/delete and active file events.

## Install
- Settings -> Extensions
- Click "Install from folder"
- Choose this folder

## Permissions
- file:read
- file:write
- ui:commands
- ui:panels

## Verify
- Enable plugin system
- Trust the plugin
- Enable the plugin
- Open Command Center (Ctrl+K)
  - Run "Obsidian Compat: Create Demo File"
  - Run "Obsidian Compat: Rename Demo File"
  - Run "Obsidian Compat: Delete Demo File"
  - Run "Obsidian Compat: Watch Active File (logs)"
  - Run "Obsidian Compat: Watch Vault Events (logs)"

The watch commands log changes for 30 seconds.
