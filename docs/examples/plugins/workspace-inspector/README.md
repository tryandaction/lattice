# Workspace Inspector (Example)

This example demonstrates workspace and annotation APIs.

## Install
- Settings -> Extensions
- Click "Install from folder"
- Choose this folder

## Permissions
- file:read
- annotations:read
- ui:commands
- ui:panels

## Verify
- Enable plugin system
- Trust the plugin
- Enable the plugin
- Open Command Center (Ctrl+K)
  - Run "Workspace Inspector: List Files"
  - Run "Workspace Inspector: List Annotations (first file)"
- Open Plugin Panels (Ctrl+Shift+P)
  - Open "Workspace Inspector"
