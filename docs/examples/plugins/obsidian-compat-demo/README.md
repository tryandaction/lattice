# Obsidian Compat Demo (Example)

This example uses the Obsidian compatibility shim exposed on ctx.obsidian.

## Install
- Settings -> Extensions
- Click "Install from folder"
- Choose this folder

## Permissions
- file:read
- ui:commands
- ui:panels

## Verify
- Enable plugin system
- Trust the plugin
- Enable the plugin
- Open Command Center (Ctrl+K)
  - Run "Obsidian Compat: List Markdown Files"
  - Run "Obsidian Compat: Active File"
