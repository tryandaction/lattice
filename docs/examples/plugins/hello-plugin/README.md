# Hello Plugin (Example)

This folder is a ready-to-install example plugin package.

## Install
- Settings -> Extensions
- Click "Install from folder"
- Choose this folder

## Verify
- Enable plugin system
- Trust the plugin
- Enable the plugin
- Open Command Center (Ctrl+K) and run "Hello Demo: Say Hello"
- Open Plugin Panels (Ctrl+Shift+P) and open "Hello Panel" or "Hello Panel (Runtime)"
