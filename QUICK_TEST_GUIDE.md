# 🚀 Quick Test Guide - Critical Bugs Fix

## 快速测试步骤

### 1️⃣ 启动应用
```bash
npm run dev
# 或
npm run build && npm start
```

### 2️⃣ 打开测试文件
在应用中打开这些文件：
- `test-100-lines.md` ✅ 应该看到 100 行
- `test-500-lines.md` ✅ 应该看到 500 行
- `test-all-bugs.md` ✅ 综合测试

### 3️⃣ 检查控制台
按 F12 打开开发者工具，查看：
- `[parseDocument]` - 解析日志
- `[FileSwitch]` - 文件切换日志
- `[buildDecorations]` - 装饰器构建日志

### 4️⃣ 验证修复

#### Bug #1: 长文件 ✅
- 滚动到文件底部
- 应该看到 "END" 标记
- 控制台显示完整行数

#### Bug #2: 文件切换 ✅
- 切换不同文件
- 内容应该立即更新
- 控制台显示 `[FileSwitch]` 日志

#### Bug #3: 文本重复 ✅
- 粗体只显示一次（不是 `**text**`）
- 斜体只显示一次（不是 `*text*`）
- 链接只显示文本（不是 `[text](url)`）

#### Bug #4: 公式渲染 ✅
- $E=mc^2$ 应该正确渲染
- 表格中的公式应该正确
- 不应该看到 "undefined"

#### Bug #5: 语法标记 ✅
- 标题的 `#` 应该隐藏
- 粗体的 `**` 应该隐藏
- 移动光标时标记出现

## 🐛 如果发现问题

1. **查看控制台日志** - 寻找错误或警告
2. **记录重现步骤** - 详细描述如何触发问题
3. **截图** - 如果有视觉问题
4. **复制日志** - 相关的控制台输出
