# 开发路线图

## 当前版本 v0.2.0

### 已完成 ✅

- PDF 阅读与批注（高亮、区域、文字批注、评论）
- Markdown 实时预览编辑
- 手写笔记（压感、多笔刷、图层）
- Jupyter Notebook 支持
- 多格式文件预览
- 桌面应用（Windows）
- 默认文件夹设置
- 多语言支持（中/英）
- 主题切换
